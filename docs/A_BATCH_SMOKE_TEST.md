## 必測 1 ─ 月結鎖定 結算流程 ⏱ 5 分鐘

1. 側欄點 **「薪資計算」**
2. 月份選一個有資料的舊月份，例如 **2026-04**
3. 看標題右上角，應該有一顆 🔐 **「結算並鎖定」** 紅色按鈕
4. 點下去 → 跳出對話框，**內容應顯示：員工數 ＋ 應發總額**
5. 按「確定」
6. ✅ **預期：**
   - 跳出「已鎖定 2026-04」訊息
   - 右上角徽章變成 🔒 **「已鎖定（系統管理員 ‧ 2026-05-XX）」**
   - 「結算並鎖定」按鈕消失，改出現 🔓 **「解鎖」** 按鈕

**結果：** [ ] ✅ 通過　　[ ] ❌ 失敗 → 把看到的訊息告訴 ahn

## 必測 4 ─ 留停凍結影響餘額 ⏱ 5 分鐘

1. 側欄點 **「員工管理」**
2. 選一個**在職員工**（找一個容易記的），點編輯
3. **先記下他現在的特休天數**（員工 dashboard → 假別餘額，或請該員工 Line 告訴你）
4. 編輯視窗應該有一個 **「設定留停期間」勾選方塊**（位置在表單下半部）
5. 勾選後：
   - 起始日填 **`2024-07-01`**
   - 結束日**留空**
6. 按儲存
7. 登出（或請該員工登入）→ 進 **「假別餘額」**
8. ✅ **預期：**
   - 該員工的特休天數**比第 3 步記下的天數少**（留 6 個月 → 大概少 6 個月年資 → 特休天數會降）
9. 側欄看「系統日誌」應有一筆 **「設定留停」**

**測完請恢復：** admin 編輯該員工 → **取消勾選**「設定留停期間」→ 儲存。系統日誌會多一筆「清除留停」。

**結果：** [ ] ✅ 通過　　[ ] ❌ 失敗 → 告訴 ahn 哪邊沒看到

## 必測 5 ─ 出勤紀錄 PDF 列印 ⏱ 3 分鐘

1. 用**員工身份**登入（找一個有打卡紀錄的員工帳號）；或 admin 切「我的功能 → 我的出勤紀錄」
2. 「我的打卡紀錄」頁，選一個**有資料的月份**
3. 月份選單**右邊**應該有一顆 📥 **「列印出勤紀錄」** 綠色按鈕
4. 點下去 → 新視窗開啟、**自動跳出列印對話框**
5. 列印對話框「目的地」選 **另存為 PDF** → 存到桌面
6. 打開存下來的 PDF，✅ **檢查：**
   - 標題 **「○○○ 的出勤紀錄」**（○○○ 是員工姓名）
   - 5 個統計小卡（總筆數 / 總工時 / 正常 / 遲到 / 早退）
   - 明細表格完整（每筆日期、上下班時間、工時、狀態）
   - 頁尾 **「列印時間：... ｜ 嘉義青年實驗室打卡系統」**
   - 紙張是 **A4**，表格沒被切到一半

**結果：** [ ] ✅ 通過　　[ ] ❌ 失敗 → 告訴 ahn 哪邊不對
